# SSN Validator
### 
Author: 
> Joan Ogando 

## Description
Small function tu validate SSN numbers.

## Installation 

You need to install node to run installation commands 

1. Clone the repository

2. Run 
```
npm install
```

3. Build
```
npm run build
```

## Usage

Run
```
node index.js <args>
```

## Test

Run
```
npm run test
```
